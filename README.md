部分功能:
- 手机验证码登录注册,以及修改<br>
- 添加好友 (包括一些空账号,已是好友的判断)<br>
- 删除好友 (包括清除好友关系以及聊天记录)<br>
- 好友请求审核<br>
- 好友详细信息展示<br>
- 消息未读提醒<br>
- 聊天记录存储<br>
- 心跳机制,读写超时处理<br>
- 资料修改,头像云oss存储<br>
- 投诉反馈<br>
- 所有能即时的我这里基本都实现了即时效果(包括通讯,好友请求,添加,删除等等)<br>
